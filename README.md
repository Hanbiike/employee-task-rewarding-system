# AEA System - Система управления персоналом, проектами и KPI-вознаграждениями

## Описание проекта

AEA System - это комплексная веб-система для управления персоналом, проектами и автоматического расчёта KPI-вознаграждений. Система предназначена для организаций с иерархической структурой (CEO → Менеджеры → Сотрудники).

## Основные возможности

### Для CEO
- Управление всеми проектами организации
- Назначение проектов менеджерам
- Управление KPI показателями
- Просмотр аналитики по всей организации
- Управление сотрудниками и отделами
- Расчёт и распределение вознаграждений

### Для Менеджеров
- Управление проектами своего отдела
- Создание и назначение задач сотрудникам
- Оценка KPI подчинённых
- Мониторинг выполнения задач
- Управление командой

### Для Сотрудников
- Просмотр назначенных задач
- Обновление статуса выполнения
- Просмотр личных KPI
- Отслеживание вознаграждений

## Технологии

- **Backend:** PHP 8+ (ООП, PDO)
- **Database:** MySQL 8+
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Security:** Password hashing, CSRF protection, Prepared statements

## Установка

### Требования
- PHP 8.0 или выше
- MySQL 8.0 или выше
- Apache/Nginx с mod_rewrite
- MAMP/XAMPP/LAMP (для локальной разработки)

### Шаги установки

1. **Клонируйте проект в директорию веб-сервера:**
   ```bash
   cd /Applications/MAMP/htdocs/
   # Проект уже должен быть в папке employee-rewarding-system
   ```

2. **Создайте базу данных:**
   - Откройте phpMyAdmin (http://localhost:8888/phpMyAdmin)
   - Выполните SQL-скрипт из файла `structure.sql`

3. **Настройте подключение к БД:**
   - Откройте файл `config/database.php`
   - Проверьте параметры подключения:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'AEA_DB');
   define('DB_USER', 'root');
   define('DB_PASS', 'root'); // Ваш пароль MAMP
   ```

4. **Инициализируйте тестовые данные:**
   - Откройте в браузере: http://localhost:8888/employee-rewarding-system/install.php
   - Скрипт создаст отделы, пользователей и базовые KPI показатели

5. **Войдите в систему:**
   - Откройте: http://localhost:8888/employee-rewarding-system/
   - Используйте тестовые учётные данные:
     - **CEO:** ceo@aea-system.com / ceo123
     - **Manager:** manager1@aea-system.com / manager123
     - **Employee:** employee1@aea-system.com / employee123

## Безопасность

- Все пароли хешируются с использованием `password_hash()` (bcrypt)
- Защита от SQL-инъекций через подготовленные запросы (PDO)
- CSRF-защита для форм
- Проверка прав доступа на уровне ролей
- Сессии с таймаутом

## Работа с KPI

### Создание KPI показателя (CEO)
1. Перейдите в раздел "KPI"
2. Создайте новый показатель (название, вес, целевое значение)
3. Показатель станет доступен для всех сотрудников

### Установка значений KPI (Менеджер)
1. Перейдите в "KPI команды"
2. Выберите сотрудника и период
3. Введите фактические значения показателей

### Расчёт вознаграждения
Формула: **Вознаграждение = Базовая_зарплата × (1 + KPI_total)**

Где KPI_total рассчитывается как взвешенная сумма выполнения всех показателей.

## Особенности реализации

- **ООП подход:** Все основные функции реализованы в классах
- **Singleton для БД:** Единое подключение к базе данных
- **Транзакции:** Использование транзакций для сложных операций
- **Роли:** Гибкая система прав доступа (CEO, Manager, Employee)
- **Адаптивный дизайн:** Современный UI с использованием CSS Grid/Flexbox
